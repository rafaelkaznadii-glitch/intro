## Publish with GitHub Pages

### Personal site

1. Create a public repository named `YOUR-GITHUB-USERNAME.github.io`.
2. Upload everything in this folder to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`, then save.

The site will be available at `https://YOUR-GITHUB-USERNAME.github.io/`.

### Project site

You can also use any repository name. Follow the same steps and the site will be published at:

`https://YOUR-GITHUB-USERNAME.github.io/REPOSITORY-NAME/`

All asset paths are relative, so both publishing options work.

## Edit the content

- Main page: `index.html`
- Design and responsive layout: `styles.css`
- Theme, navigation and project galleries: `script.js`
- Images and PDF documents: `assets/`
